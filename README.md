## Features

- **Next.js Integration**  
  The portfolio is built using Next.js, leveraging its powerful features for server-side rendering, static site generation, and seamless routing.

- **TypeScript Support**  
  TypeScript is used throughout the application to ensure type safety and enhance code reliability and maintainability.

- **Prisma ORM**  
  Prisma is utilized for efficient database management and querying, providing a robust backend for handling data operations.

- **Zod for Schema Validation**  
  Zod is employed for schema validation, ensuring that data adheres to the expected format and improving error handling.

- **Axios for Data Fetching**  
  Axios is used to make HTTP requests and fetch data from APIs, providing a reliable and straightforward way to handle asynchronous operations.

- **Responsive Design**  
  Tailwind CSS ensures a fully responsive design, making the portfolio accessible and visually appealing on all devices.

- **Project Showcases**  
  Detailed sections highlighting various projects, demonstrating expertise in web development technologies.

- **Resume Download**  
  A feature to download my resume for a complete overview of my professional experience and skills.

- **Contact Form**  
  An easy-to-use contact form allowing visitors to get in touch with me for inquiries or messages.

- **Blogs**  
  A section for blog posts where you can read about my development journey, industry insights, and technical tutorials.

## Run Locally

Clone the project

```bash
  git clone https://github.com/SazzatTuhin/portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DATABASE_URL`

`NEXT_PUBLIC_PIN`
